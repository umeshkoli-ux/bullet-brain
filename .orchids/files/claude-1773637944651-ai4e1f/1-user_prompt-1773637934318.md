Create a full-stack web application in Python using Flask called Bullet Brain.

Bullet Brain is a brain wellness web app that helps users improve focus, relax, sleep better, and meditate through audio sessions.

Requirements:

Backend

Use Python Flask as the backend framework.

Use MySQL database to store user accounts.

Implement user authentication (register, login, logout).

Use Flask sessions to keep users logged in.

Protect pages so users must log in before accessing them.

Project Structure
The project should follow this structure:

app.py (Flask backend)

templates/

login.html

register.html

home.html

enter.html

focus.html

relax.html

sleep.html

meditation.html

chatbot.html

static/

audio/

css/

images/

Login System
Create a login page where users enter email and password.

Create a registration page where users can sign up with:

username

email

password

Store users in a MySQL table called users.

Home Page
After login the user should see a home page with:

Two options:

Enter

Ask

Enter → manually choose mood
Ask → open AI chatbot

Enter Page
The Enter page should allow users to choose:

Focus

Relax

Sleep

Meditation

Each option opens a page containing multiple audio players.

Example:
Focus page → focus audio sessions
Relax page → relaxation sessions
Sleep page → sleep sounds
Meditation page → meditation sessions

All audio files should be loaded from the static/audio folder.

Audio Player
Each session page should contain multiple HTML5 audio players so users can play audio directly in the browser.

Chatbot Page (Ask Feature)
Create a chatbot interface where users can type how they feel.

The chatbot should ask questions such as:

How are you feeling today?

Are you stressed, distracted, tired, or anxious?

Based on user responses, recommend one of the four moods:

Focus

Relax

Sleep

Meditation

Then redirect the user to the appropriate session page.

UI Design
Use a dark modern UI theme.

Include:

cards for sessions

back arrows for navigation

profile icon on the home page

responsive layout

Navigation
User flow:

Login → Home → Enter → Mood Page → Audio Session

or

Login → Home → Ask → Chatbot → Recommended Session

Code Quality

Clean Flask routes

Organized templates

Use url_for() for static files

Include comments explaining the code

The result should be a fully functional Flask application that can run locally using python app.py.
