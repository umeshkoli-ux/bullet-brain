# AI-Based Clinical Decision Support System (CDSS)

## 📌 Overview
This project is a web-based Clinical Decision Support System (CDSS) designed to analyze user inputs such as focus levels, sleep patterns, and stress indicators to provide intelligent recommendations.

## 🎯 Objective
To assist in decision-making related to mental well-being by simulating how healthcare systems process patient data and generate insights.

## 🧠 Features
- Focus level analysis
- Sleep pattern evaluation
- Stress and relaxation monitoring
- AI-based recommendation system

## ⚙️ Tech Stack
- Frontend: HTML, CSS
- Backend: Python (Flask)
- Deployment: Vercel

## 🤖 AI Approach
The system uses rule-based decision logic to process multiple inputs (focus, sleep, stress) and generate recommendations. This simulates the working of a Clinical Decision Support System (CDSS).

Future improvements:
- Integration of Machine Learning models (Decision Trees, Logistic Regression)
- Real-world healthcare dataset usage

This project demonstrates how AI concepts can be applied in healthcare systems to assist in decision-making and improve user well-being.
